# Webtop App Theme
Styles and Themes for Webtop applications.

## Installation

```sh
npm install --save-dev @mintjamsinc/webtop-app-theme
```

## Usage

```vue
<style lang="scss">
@import "@mintjamsinc/webtop-app-theme/css/style.scss";
</style>
```
